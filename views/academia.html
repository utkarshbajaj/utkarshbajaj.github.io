Last semester, I took a course on distributed systems, where I learned the fundamentals of building reliable and scalable systems. Introduction to Computer Systems (18613)

In my Computer Systems class, I delved into a class on low-level computer concepts like assembly, cache, memory allocation, and more. It was incredibly challenging but also deeply rewarding. I gained insights into how things actually work under the hood, moving beyond the higher-level programming perspective to understand how code truly runs on a machine. The hands-on labs, such as building a cache simulator, implementing memory allocation, and creating a tiny shell, were particularly engaging and sparked a strong interest in coding in C. This experience has inspired me to explore more C projects and further develop my skills in low-level programming.

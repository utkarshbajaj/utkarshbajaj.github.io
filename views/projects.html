Codeforces Rating Updates

Developed an application that automates the delivery of Codeforces contest rating updates. The project uses Google Firebase to manage user data and AWS EC2 to execute Python scripts that handle the updates. I built the user interface with HTML, CSS, and JavaScript and hosted it on GitHub Pages.

Once a contest's ratings are published, the application sends an email notification to users, saving them the hassle of manually checking the Codeforces website. This project made it easier for users to stay updated on their contest performance. I'm currently working on releasing this to production so that contestants can start using it.

Find IMDB Ratings

Created a Python script during Hacktoberfest 2020 using BeautifulSoup, requests, and pandas to scrape the web for movie data and store it in a .csv file for easy analysis. The project attracted multiple forks and contributions from over 5 individuals on GitHub, turning into a collaborative effort. This script made it simple for users to gather and analyze movie data in a structured format.

Emergency Social Network

As part of my Software Engineering course, I developed an Emergency Social Network application using a stack of vanilla JavaScript, HTML, and CSS. The application included features such as public and private chatting, posting announcements, and navigating to nearby shelters displayed on a map. Through this project, I also learned how to effectively apply a design pattern in the application code, ensuring a structured and organized approach to solving complex problems.
